Oh - one more thing:
A user named "Arclight" pointed out that Cart Life won't run correctly if located in Windows 7's "Program Files (x86)" directory.
It'll say "Unable to write to the current directory (...)" and crash. So if you're getting this crash for this reason, just move the whole thing elsewhere on your disk and it'll be fine. I mean, I hope it will.
